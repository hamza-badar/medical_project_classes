Title - Medical Shop Desktop Application.

Langugaes Used  = Java with Swing, AWT and Applets.

Database Used - SQLite.

Discription - This is a fully functional cross platform desktop application in which owner can perform complete CRUD operations like Add Medicines , 
Delete Medicines , Update Medicines , Check Medicines , Update Price and after selecting medicines, a complete bill will be generated for future use.
